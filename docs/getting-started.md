# Getting Started

## Installation

::: code-group

```bash [pnpm]
pnpm add nanomachine
```

```bash [yarn]
yarn add nanomachine
```

```bash [npm]
npm install nanomachine
```

```bash [bun]
bun add nanomachine
```

```ts [deno]
import { createStateMachine } from "npm:nanomachine";
```

:::
